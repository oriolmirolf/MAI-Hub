## ğŸ“‚ How to Organise Your Contributions

To keep the repository clean, please follow the existing structure:

### 1. Identify the Category
Check if the subject is **Compulsory** or **Elective**.
* **`/Compulsory`**: (CI, CV, IHLT, IMAS, IML, PAR)
* **`/Elective`**: (e.g., DL, MASD, URL, NDVW...)

### 2. Folder Naming
* If the course folder already exists (e.g., `Elective/DL`), add your files inside.
* If the course folder is missing, please create it using the **official course acronym** (e.g., `AIHC`, `ATCI`).

## ğŸ”„ The Workflow

1.  **Fork** this repository.
2.  Create a new **branch** for your changes (`git checkout -b add-dl-notes`).
3.  **Add your files.**
4.  **Commit** your changes with a clear message (`git commit -m "Add my repo for Deep Learning 2026"`).
5.  **Push** to your branch (`git push origin add-dl-notes`).
6.  Open a **Pull Request (PR)** against the `main` branch.
